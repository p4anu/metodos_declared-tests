ABOUT "metodos01":

This little project is a test (exercise) to reuse and abbreviate code in the class. And it is also a simple form of branching. 
The project simulates a car registration program. Among the data in this registry, the variable "ID" is declared as static (static int) to operate an automatic increment of its value on each entry made by the user. The user can register as many vehicles as he wants until he decides not to continue (in the code this uses a do while loop).
This project contains its .jar file
Test project created by Miguel Ángel De Zan (p4anu)

SOBRE “metodos01”:

Este pequeño proyecto es un ensayo (ejercicio) para reutilizar y abreviar código en la clase <main>. Y es también una sencilla forma de ramificación. 
El proyecto simula un programa de registro de automóviles. Entre los datos de este registro, la variable “ID” se declara como estática (static int) para operar un incremento automático de su valor en cada ingreso que hace el usuario. El usuario puede registrar todos los vehículos que quiera hasta que decide no continuar (en el código esto utiliza un bucle do while).
Este proyecto contiene su archivo .jar
Proyecto de prueba creado por Miguel Ángel De Zan (p4anu)
